kagenomise.cart Installation
----------------------------

To install kagenomise.cart using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``kagenomise.cart`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        kagenomise.cart

* Re-run buildout, e.g. with:

    $ ./bin/buildout
